#  Customer Performance Analytics Dashboard

An interactive Business Intelligence dashboard built using Power BI to analyze customer demographics, revenue contribution, and purchasing behavior.

This project demonstrates advanced DAX logic, dynamic insight generation, and executive-ready reporting design.

---

##  Project Overview

This dashboard transforms transactional customer data into meaningful business insights. It provides clear visual analysis of revenue distribution, demographic segmentation, and performance trends.

The objective was to design a professional, interactive dashboard that supports data-driven decision-making.

---

##  Key Insights Delivered

- Revenue distribution by age group  
- Gender-based revenue segmentation  
- Customers with and without children analysis  
- Top-performing country identification using TOPN  
- Percentage contribution analysis  
- Automated narrative insights using dynamic DAX measures  

---

##  Tools & Technologies Used

- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Bookmark Navigation  
- Conditional Formatting  
- Interactive UI Design  

---

##  Technical Highlights

- Use of `VAR` for structured DAX logic  
- Context manipulation using `CALCULATE` and `ALL()`  
- Ranking with `TOPN()`  
- Percentage calculations using `DIVIDE()`  
- Dynamic caption generation with formatted text outputs  
- Light/Dark theme switching using bookmarks  

---

##  Dashboard Features

- Interactive KPI cards  
- Donut charts for gender segmentation  
- Age-group revenue comparison  
- Reference lines for average performance  
- Bookmark-based theme toggle (Light/Dark mode)
